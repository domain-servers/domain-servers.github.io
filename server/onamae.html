お名前レンタルサーバーはドメインシェアNo1の「お名前.com」の提携レンタルサーバーになります。 2019年にサービスをリニューアルし、スペックやサービス内容が大幅に変わりました。

2019年以降、毎年着実にバージョンアップを重ね最近ではデメリットだった高速化の部分も改善されてきました。

さらにには国内の共用レンタルサーバーでは初となる「WEXAL（ウェクサル）/ Page Speed Technology」を標準搭載。 これはCONOHA WIINGとお名前レンタルサーバーだけが導入しているプライムテクノロジー社の技術になっています。

サービス面ではSLA稼働率保障99.99％。ワードプレス簡単インストール。かんたん引越し。テスト環境構築。ドメイン永久無料。無休の電話サポートなど充実したサービスを提供。 データ容量も400GBとかなりの大容量になっています。

サービス面においては業界No1と言っても言い過ぎではないぐらい充実したプラン内容になっています。 テスト環境構築や無休サポートなどはエックスサーバーやConohaWINGでも行っていません。

これまでに37万人がお名前レンタルサーバーを愛用。ドメイン会社であるお名前.comと一緒に契約するとネームサーバー設定が簡易化されます。

高速化に配慮したサーバースペック

お名前サーバーのサーバー環境は「Webサーバー」「ストレージ」「高速化」の3つの基本環境について、 それぞれ施策を行っています。

  • 安定：Webサーバーは同時アクセスの処理に優れたNginx（エンジンエックス）
  • 安定：RAID10での管理体制
  • 高速化：ストレージはソリッドストレージ
  • 高速化：「WEXALR Page Speed Technology」
  • 高速化：HTTP/2対応
  • 高速化：超高速アプリケーション実行環境(LSAPI)
  • 高速化：コンテンツキャッシュ

まず、Webサーバーソフトは「Xginx」でストレージはソリッドストレージとなります。高速化で重要な2つの要素についてはクリアしています。

高速化技術では「LSAPI」「コンテンツキャッシュ」「HTTP2」が導入されていて他の高速化対応されているレンタルサーバーとあまり変らない内容になっています。 さらにAI技術である「WEXAL」が導入されたことにより大幅なスピードアップを実現。サーバー処理速度が業界最高峰となりました（2020年12月自社調べ）。

データ容量は400GBにグレードアップ

データ容量は400GBと充分な容量が用意されています。お名前サーバーには1つのプランしかないため、容量は400GBで固定されます。

サービス面が業界最高水準！

（１）稼働率は99.99％で安定

稼働率はSLA保証制度として99.99％を数値として設定しています。 これは、稼働率が規定数値（99.99％）を下回った月は一定金額の返金をしてくれる制度になります。

また、実際の稼働率としても99.99％を実現させています（2021年度6月調べ）、かなり安定して管理されているレンタルサーバーだと言えます。

（２）ワードプレス簡単インストールが本当にかんたん

エックスサーバーやConohaWingと同様にワードプレスの構築が本当に簡単になっています。

「タイトル」「名前」「パスワード」「URL」などと、わずか5つほどの入力を行うだけでワードプレスが作れます。

（３）夜でも土日でも電話できる。無休の電話サポート

お名前レンタルサーバーで特に魅力的な部分と言えるのが、電話サポートの対応時間の長さになります。

平日も休日・祝日も無休で電話サポートを受け付けています。 当サイトで紹介している会社では最も電話の対応時間が長いサービスになっています。

多くの競合他社はコスト削減のため、電話サポートは平日だけ行っているところが多くなっています。 土日も電話を受け付けてくれるため、設定関係が不得意な方や、土日に作業を行いたい方にとっては、 喜ばしいサポート体制になっています。

（４）テスト環境の構築ができる

ワードプレスの制作にあたり、テスト環境を作って管理することが出来ます。 ワードプレスのバージョンアップ時やプラグインを導入する際にテスト環境で確認すれば、エラーが出てもすぐに戻せるため、エラーを回避することが出来ます。

共用サーバーにおいてこのテスト環境のサービスを実施しているのは「CPI」「さくらインターネット」の2社だけになります。

（５）サーバー環境のチェック・変更ができる

さらに独自のサービスとして、割当てられているサーバーの混雑チェックが出来ます。チェックできるだけでなく、割当サーバーの変更も可能になっています。 このサービスも他社では「ConohaWING」だけが行っているサービスになります。

このように、サービスとサポート面では競合他社よりも内容が充実しているだけでなく、提供されているサービスの種類も多くなっています。

唯一、「設定のお任せサポート」については無料、有料かかわらず行っていませんが（これは法人向けのエックスサーバービジネスで無料提供）、 ほぼ共用サーバーにおいてサービス内容No1と言っていいぐらいの内容になっています。

料金について

初期費用が0円＋月額料金「初月と翌１ヶ月分が無料」なので大変お得

お名前サーバーはプランが1つだけになので料金体系もシンプルになっています。初期費用は0円で、基本の月額料金が1,430円になります。 12ヶ月払いで契約すると長期割引によって月額1,078円まで料金がさがります。

特徴的なのはお申込み月と、その翌月分の月額料金が0円になる点。初年度については最長で約2ヶ月の月額料金が無料になります。

メリットとデメリット

【メリット】総合的なサービスの良さ

2019年のリニューアルによってサーバー環境やサービス内容が大幅改善されたお名前レンタルサーバー。

充実した電話サポート。安定した稼働率や、他社ではあまり行っていないサービスの提供。そしてオリジナルの管理画面。

ベスト3のレンタルサーバーとして紹介はしていませんが、サービス内容で考えれば共用サーバーBEST3には間違いなく入ります。

【デメリット】毎年のバージョンアップでデメリットが減ってきている

これまでは高速化対応の面において、あまり速くないとされてきたお名前レンタルサーバーではありますが、WEXALの導入やその他の高速化施策によって かなり速いプランになりつつあります。そのためデメリットらしいデメリットがないプランになっています。
